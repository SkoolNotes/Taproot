Kirkoff’s Laws

Kirkoff’s First Law

Definition \(\cdot\) [ Kirkoff’s First Law *Sum of voltage in any closed loop should add up to 0** ]

As in, the sum of all voltage changes from Start => Start will add up to 0.

Kirkoff’s Second law

Definition \(\cdot\) [ Kirkoff’s Second Law *Net current flowing into a node is 0** ]

With a current \(i_0\), when it flows into a junction like B, the current \(i_0\) splits into \(i_2\) and \(i_3\)
