feynmann diagrams

start with something like a spacetime diagram

  1. the path tells us everything there is to know about a particle

  2. but at events (ex. where the path changes directions) any number of things might happen

  3. subatomic events seem to be instant and unsurvivable (nothing goes in and also comes out)

features of the feynmann diagram

  1. every vertex has three points, with two fermion lines (particles) and one boson line (force carrier)

    1. every interaction in the universe is the result of two fermions and a boson meeting in these triads

  2. arrows denote anti-ness (a forwards-in-time positron is the same as a backwards-in-time electron)

  3. conservation laws are consistent at each vertex

  4. composite particles don’t interact with bosons, so we draw them as their constituent quarks
